# JavaScript Assignment

A collection of beginner-friendly JavaScript projects built to practice core JavaScript concepts, DOM manipulation, event handling, and problem-solving skills.

## 📌 About This Repository

This repository contains 14 mini JavaScript projects that cover fundamental web development concepts. Each project is designed to help learners gain hands-on experience with JavaScript and build interactive web applications.

## 📂 Project Structure

```text
Javascript-assignment/
│
├── project-01-Counter App

├── project-03-Character-counter

```

## 🚀 Projects Included

### 01. Counter App

* Increase counter
* Decrease counter
* Reset counter

### 02. Background Color Changer

* Change background color dynamically
* Practice event handling

### 03. Character Counter

* Count typed characters in real time

### 04. Age Calculator

* Calculate age based on date of birth

### 05. Random Quote

* Display random quotes on button click

### 06. Digital Clock

* Show current time dynamically

### 07. Password Generator

* Generate secure random passwords

### 08. Gradient Background Generator

* Display CSS gradient code

### 09. Stopwatch

* Start timer
* Stop timer
* Reset timer

### 10. Number Guessing Game

* Guess a randomly generated number
* Receive feedback for each guess

### 11. Student Result Calculator

* Calculate total marks
* Calculate percentage

### 12. Simple Calculator
